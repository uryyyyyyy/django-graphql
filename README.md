
### Start app in local environment

- middleware

```
docker-compose run
```

- django

```
pipenv run manage_py runserver
```

### GraphiQL

#### Create Schema

You can dump your schema data to schema.json that is compatible with apollo-client.

```
pipenv run schema
```
